# **Finance Dashboard – Power BI**

## **Project Overview**

This project presents an interactive **Finance Dashboard** built entirely in **Power BI**, using Power Query for data cleaning and transformation.
The dashboard provides a clear view of **sales performance**, **profitability**, **discount distribution**, and **segment-level financial metrics** across multiple countries.

---

## **Dataset Summary**

* **Source:** Finance dataset
* **Total Fields Include:** Country, Segment, Product, Year, Sales Amount, Profit, Discount, COGS, Order Count
* **Purpose:** To analyze financial performance across time, geography, products, and customer segments.

---

## **Key Insights**

* **Total Sales:** 118.73M
* **Discount Given:** 9.21M
* **Total Orders:** 1M
* **Profit of Orders:** 16.89M
* **Total COGS:** 101.83M
* **Most Profitable Countries:** Germany & France
* **Top Products by Sales:** Paseo, VTT, Velo
* **Discount Distribution:** Majority in *High* and *Medium* bands
* **Profit Margin by Segment:** Channel Partners lead with the highest margin
* **Sales Trend:** Increasing pattern across multiple quarters and months

---

## **Dashboard Features**

* **KPIs:** Sales Amount, Discount, Orders, Profit of Orders, Total COGS
* **Visuals Included:**

  * Orders by Country
  * Profit Margin by Country
  * Discount by Discount Band
  * Profit Margin by Segment
  * Sales Amount by Year, Quarter, and Month
  * Sales Amount by Product
* **Slicers:**

  * **Year**
  * **Segment**
  * **Country**

---

## **Tools Used**

* **Power BI** – Dashboard creation and modeling
* **Power Query** – Data cleaning, shaping, and transformations
* **DAX** – Calculated measures (profit margin, sales KPIs, discount %)

---

## **Project Workflow**

1. **Data Import:** Loaded raw Excel dataset into Power BI
2. **Data Cleaning (Power Query):** Fixed data types, removed blanks, standardized fields
3. **Data Modeling:** Created relationships and structured the financial model
4. **Measure Creation:** Developed DAX measures for KPIs (Sales, Profit, COGS, Discount %)
5. **Visualization:** Built interactive charts, donut visuals, KPI cards, and slicers
6. **Dashboard Design:** Clean layout with filters for intuitive financial insight analysis

---

## **Project Files**

* **Finance Dashboard POWER BI.pbix** – Main dashboard
* **Finance dataset.xlsx** – Raw data

---

## **How to Use**

1. Download all files from the repository
2. Open **Finance Dashboard POWER BI.pbix** in **Power BI Desktop**
3. Ensure **Finance dataset.xlsx** stays in the same folder so the data source connects correctly
4. Use slicers (**Year, Segment, Country**) to explore insights dynamically
5. Refer to **FINANCE PBI.png** for a quick preview of the dashboard
