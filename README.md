# Finance-Dashboard----Power-BI

## **Project Overview**

This project presents an interactive Finance Dashboard built entirely in Power BI, using Power Query for data cleaning and transformation.
The dashboard provides a clear view of sales performance, profitability, discount distribution, and segment-level financial metrics across multiple countries.

## **Key Insights**

⦁	Total Sales: 118.73M

⦁	Discount Given: 9.21M

⦁	Total Orders: 1M

⦁	Profit of Orders: 16.89M

⦁	Total COGS: 101.83M

⦁	Most Profitable Countries: Germany & France

⦁	Top Products by Sales: Paseo, VTT, Velo

⦁	Discount Distribution: Majority in High and Medium bands

⦁	Profit Margin by Segment: Channel Partners lead with the highest margin

⦁	Sales Trend: Increasing pattern across multiple quarters and months

## **Dashboard Features**

KPIs: Sales Amount, Discount, Orders, Profit of Orders, Total COGS

Visuals Included: 

⦁	Orders by Country

⦁	Profit Margin by Country

⦁	Discount by Discount Band

⦁	Profit Margin by Segment

⦁	Sales Amount by Year, Quarter, and Month

⦁	Sales Amount by Product

Slicers:

⦁	Year

⦁	Segment

⦁	Country

## **Tools Used**

⦁	Power BI – Dashboard creation and modeling

⦁	Power Query – Data cleaning, shaping, and transformations

⦁	DAX – Calculated measures (profit margin, sales KPIs, discount %)

## **Project Workflow**

⦁	Data Import: Loaded raw Excel dataset into Power BI

⦁	Data Cleaning (Power Query): Fixed data types, removed blanks, standardized fields

⦁	Data Modeling: Created relationships and structured the financial model

⦁	Measure Creation: Developed DAX measures for KPIs (Sales, Profit, COGS, Discount %)

⦁	Visualization: Built interactive charts, donut visuals, KPI cards, and slicers

⦁	Dashboard Design: Clean layout with filters for intuitive financial insight analysis

## **Project Files**

⦁	Finance Dashboard POWER BI.pbix – Main dashboard

⦁	Finance dataset.xlsx – Raw data

## **How to Use**

⦁	Download all files from the repository

⦁	Open Finance Dashboard POWER BI.pbix in Power BI Desktop

⦁	Ensure Finance dataset.xlsx stays in the same folder so the data source connects correctly
